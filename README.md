# 🏥 Chikitsak-AI

AI-powered healthcare assistant providing smart, user-centric solutions for both patients and doctors.

---

## 🚀 Features

### 👤 For Patients
- Symptom Checker  
- Mental Health Support  
- First Aid Assistant  
- Video Consultation  
- Appointment Booking  
- Health Analytics  
- Emergency Assistance  

### 🩺 For Doctors
- Manage patient appointments  
- Access patient history & reports  
- Add prescriptions & medical notes  
- Medication management system  

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication & APIs:** Supabase  

---

## ⚙️ Installation

```bash
git clone https://github.com/rajveer-verma/Chikitsak-AI.git
cd Chikitsak-AI
npm install
npm run dev

---
